QUESTION:

4. The table shows the delays (pico second) of the components in Figure 4.17.

Calculate the delay of the 4 types of instructions. You need to show which components are used in each instruction.

ANSWER:

(1)Instruction ADD
=> (PC Read + I-mem + Mux + Register file + Mux + ALU + Mux + Register file)
=> (10 + 250 + 25 + 150 + 25 + 200 + 25 + 150) = 835 picoseconds
---------------------------------------------------------------------------
(2)Instruction LD
=> (PC Read + I-mem + Mux + Register file + Mux + ALU + D-mem + Mux + Register file)
=> (10 + 250 + 25 + 150 + 25 + 200 + 250 + 25 + 150) = 1085 picoseconds
---------------------------------------------------------------------------
(3)Instruction SD
=> (PC Read + I-mem + Mux + Register file + Mux + ALU + D-mem)
=> (10 + 250 + 25 + 150 + 25 + 200 + 250 ) = 910 picoseconds
---------------------------------------------------------------------------
(4)Instruction BEQ
=> (PC Read + I-mem + Mux + Register file + Mux + ALU + And + Mux + PcWrite)
=> (10 + 250 + 25 + 150 + 25 + 200 + 10 + 25 + 10) = 705 picoseconds
