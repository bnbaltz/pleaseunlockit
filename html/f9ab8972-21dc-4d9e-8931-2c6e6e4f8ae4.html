QUESTION:

In Java - Purpose:
Understand I/O using Streams and Files

Description:
A college keeps records of all professors and students in a file "people.txt". Each line starts with a title (Professor/Student) followed by first name, last name, and a department for a professor or a degree for a student as follows:

"people.txt":
Professor Blake Paul Science
Student Mike Smith Math
Professor Brian McGinty SocialWork
Student Rodney James History
Professor Blaise Pascal Physics
Student Julia Roberts Masters
Professor Feida Nyla Philosophy
Student Kyle Turpin Geology

Create two classes: Professor and Student with applicable fields.
Write a program that will read from file "people.txt", scan every line, create the necessary object (either a student or professor) and add it to one of the two arrays ( or lists): students and professors.

Once the lists/arrays are ready:
- print both to standard output
- serialize them using object streams into "professors.ser" and "students.ser"

ANSWER:

Code:

import java.io.*;
import java.util.*; 

//class Professor to hold data for professor
class Professor implements Serializable{
  public String fname; 
	public String lname; 
  public String dept; 
}

//class Student to hold data for student
class Student implements Serializable{
  public String fname; 
	public String lname; 
  public String dept; 
}

class Main {
  public static void main(String[] args){
    //define list of professor and student for objects
    List<Professor> prof =new ArrayList<Professor>();  
    List<Student> stu =new ArrayList<Student>();
    
    try  
    {  
      File file=new File("people.txt");    //file instance  
      FileReader fr=new FileReader(file);       //read  file
      BufferedReader br=new BufferedReader(fr); //character input stream        
      String line;     //variable for holding containt of line

      //read file line by line 
      while((line=br.readLine())!=null)  
      {  
        //split line with space
        String[] data = line.split(" ", 4);   

        //check title and create appropriate object and add to list 
        if(data[0].equals("Professor")){
          Professor p = new Professor();
          p.fname = data[1];
          p.lname = data[2];
          p.dept = data[3];

          prof.add(p);
        }else{
          Student s = new Student();
          s.fname = data[1];
          s.lname = data[2];
          s.dept = data[3];

          stu.add(s);
        }
      } 
     
      //close file 
      fr.close();            
    }  
    catch(IOException e)  
    {  
      e.printStackTrace();  
    } 

    //display professor list 
    System.out.println("Professor List : ");
    for(Professor p:prof){
      System.out.println("\t"+p.fname +" "+ p.lname +" "+ p.dept);
    }

    //display student list 
    System.out.println("\nStudent List : ");
    for(Student s:stu){
      System.out.println("\t"+s.fname +" "+ s.lname +" "+ s.dept);
    }

    //serialize professor list with name "professors.ser"
    try
		{ 
			FileOutputStream file = new FileOutputStream("professors.ser"); 
			ObjectOutputStream out = new ObjectOutputStream(file); 
			
			//method for serialization of object  
			out.writeObject(prof); 
			
      //close both file
			out.close(); 
			file.close(); 
			
			System.out.println("\nProfessor has been serialized"); 
		} 		
		catch(IOException ex) 
		{ 
			System.out.println("IOException is caught in serializing Professor list."); 
		}

    //serialize professor list with name "students.ser"
    try
		{ 
			FileOutputStream file = new FileOutputStream("students.ser"); 
			ObjectOutputStream out = new ObjectOutputStream(file); 
			
			//method for serialization of object 
			out.writeObject(stu); 
			
      //close both file
			out.close(); 
			file.close(); 
			
			System.out.println("Student has been serialized"); 
		} 		
		catch(IOException ex) 
		{ 
			System.out.println("IOException is caught in serializing Student list."); 
		}
  }
}
