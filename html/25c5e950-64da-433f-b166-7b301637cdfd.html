QUESTION:

 
Two point charges q1= -88 nC and q2 = +360 nC are located atthe fixed positions x1 = 0.22m and x2 = 0.96m on the x axis.
B) Calculate a point on x axis where the electric potential =zero
C) A charge q3 , -11μC and mass = 0.0039kg released fromrest at the origin, what is the charge's final speed and directionwhen its a long distance away from the other two charges?
Please include all given values for the variables please. Nota text problem

ANSWER:

q1= -88 nC, q2 = +360 nC, x1 = 0.22 m, x2 = 0.96 m
B)
If the point is outside the charges, then
q1/(x - x1) + q2/(x - x2) = 0
-q1/(x - x1) = q2/(x - x2)
∴x = (q2*x1 + q1*x2)/(q1 + q2) = -0.019 m
If the point is between the charges, then
kq1/(x - x1) + kkq2/(x2 - x) = 0
-q1/(x - x1) = q2/(x2 - x)
∴x = (q2*x1 - q1*x2)/(q2 - q1) = 0.37 m
so there are two points.
C) q3 = -11 μC, m = 0.0039 kg
initial energy = kq3*(q1/x1 + q2/x2)
final energy = mv2/2
q3*(q1/x1 + q2/x2) = mv2/2
∴v = √[2kq3*(q1/x1 + q2/x2)/m] = 1.13 m/s
