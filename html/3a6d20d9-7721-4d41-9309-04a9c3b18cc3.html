The inputs to a project cost plan are

  • Project management plan
  • Project charter
  • Environmental factors
  • Process assets

The inputs to a project procurement plans are

  • Team agreements
  • Make/buy analysis and report
  • Requirement documents
  • Project scope baseline
  • Project schedule
  • Cost estimates
  • Resource requirements for activities/tasks
  • Cost performance baseline
  • Environmental factors
  • Process assets

We can see that there are a considerable amount of documents that are required as input for project procurement and cost plan. Most of these documents are created by discussing with various stakeholders after the requirement planning discussion.

The process groups are mentioned below and the knowledge areas that are used in each group.

  1. Initiating process group: Project integration management, Project stakeholder management
  2. Planning process group: Project integration management, Project scope management, Project schedule management, Project cost management, Project quality management, Project resource management, Project communication management, Project risk management, Project procurement management, Project stakeholder management.
  3. Executing process group: Project integration management, Project quality management, Project resource management, Project communications management, Project risk management, Project procurement management, Project stakeholder management
  4. Monitoring and control process group: Project integration management, Project scope management, Project schedule management, Project cost management, Project quality management, Project resource management, Project communication management, Project risk management, Project procurement management, Project stakeholder management.
  5. Closing process group: Project integration management

Each of these intersections of knowledge areas on the process groups provide important steps at each of the stages of project. As a result. it is important to plan these and make sure that each stage of the project is executed systematically and smoothly.
