QUESTION:

Prove unconditionally the existence of a pseudorandom function F : {0,1}* × {0,1}* →{0,1}* with l key(n) = n and l in(n) = O(logn).

Hint: Implement a uniform function with logarithmic input length.

ANSWER:

Take a simple example:

If A wants to authenticate himself to B, A can send a secret they both knows. In that case, they both can seed a Pseudo Random Number Generator(PRNG) with the secret code. For example, when A calls, B sends a random number i and A responds B by sending the ith random number generated by PRNG, thereby proving A knows the secret code sent by B. Here both A and B computes i random numbers. When B sends random i, A responds with  Fk(i) which is the PRNG of i, and PRNG Fk(r1), Fk(r2)....Fk(rn) can be generated for any random number r1,r2...rn .

A function F : {0,1}*x  {0,1}*{0,1}* with l key(n)= n and in(n)= O(logn) is a Pseudo random function.

Here the key length of PRNG is n that K= {0,1}n  and n is an integer and the input length is O(logn)

{0,1}log(n){0,1} with 2n distinct values since each input is n bits long and there are 2n entries here 

n*2n bits are needed.  For n different elements in {0,1}log(n) we have (2n)n possible functions.

 Thus the F is clearly indistinguishable from a random function and hence it is a Pseudo random function.

Note: The O(logn) function uses divide and conquer algorithm in which we need to search a subset of the entire section rather than searching the entire section . The elements on which the search is performed is n.

Example: In phone book, the O(logn) searches the phone number of the given name using their starting alphabet instead of searching the whole book.(Divide and conquer)
