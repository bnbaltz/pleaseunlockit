ANSWER:

Consider an array values. The array is of size 7.

The statement:

System.out.printIn(values[7]));//Java

Console.WriteLine(values[7]);//C#

cout << values[7] << endl;//C++

Answer: C. produce an error

  • This produces an error because we are trying to access the 7th element in the array list but actually we count the number of elements starting from 0 . So the array will have elelements from 0-6 where 6 is the 7the element. To access the 7th element, we have to use values[6] instead of values[7].

Here is the code running without an error when we access 7th element.

Code

#include <iostream>
#include <string>
using namespace std;

int main() {
  string values[7] = {"Volvo", "BMW", "Ford", "Mazda", "Benz", "Mar", "Aze"};
  cout << values[6];
  return 0;
