magnetic field , B = 5.8 *10^-5 T

a) for the south facing window 

flux passing through the window = B * A * cos(theta)

flux passing through the window 5.8 *10^-5 * 5 * cos(72 degree)

flux passing through the window = 8.96 *10^-5 T.m^2

b) for the west facing window 

the angle between area vector = B * A * cos(90 degree)

angle between area vector = 0 T.m^2
