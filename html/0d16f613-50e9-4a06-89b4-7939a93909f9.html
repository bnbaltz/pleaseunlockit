public class Main
{
   public static void main(String[] args) {
   int i=1+2*3+7*2%5;
   System.out.println("i="+i);
   }
}

output: 11

Reason: Multiplication and remainder % have the same precedence. Hence evaluation of both can be done left to right. First multiplication will be solved then remainder will be evaluated.

hence 1+2*3+7*2%5=1+6+14%5=1+6+4=11
