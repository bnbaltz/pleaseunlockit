QUESTION:

The table given below shows the lifting force on an aircraft wing measured in a wind tunnel at various wind velocities. Model the data with an interpolating polynomial of degree 5, and use that polynomial to estimate the lifting force at 2000 ft/s.

ANSWER:


%%Matlab code of interpolating polynomial of degree 5
clear all
close all

%data for velocity
vel=[1 2 4 8 16 32].*100;

%Data for lifting force
frc=[0 3.12 15.86 33.7 81.5 123].*100;

%Creating matrix for degree 5 polynomial

for i=1:6
    for j=1:6
        
        A(i,j)=vel(i)^(j-1);
        
    end
    
    b(i,1)=frc(i);
    
end

%getting solution

fprintf('Coefficient matrix for getting solution is A=\n')
disp(A)

fprintf('b vector for finding solution b=\n')
disp(b)

fprintf('Coefficient vector =\n')
disp(A\b)

vv=A\b;

syms x

%function for polynomial
f(x)=vv(1)+vv(2)*x^1+vv(3)*x^2+vv(4)*x^3+vv(5)*x^4+vv(6)*x^5;

fprintf('Displaying the function Lifting force\n')
disp(vpa(f,3))

fprintf('\n\n\tHence Lifting force at 2000 ft/s is %f \n',double(f(2000)))
