Consider the case when several persons play
the wireless phone game They send a message
from the first person until the last one and the
message gets distorted Simulate this process
according to your own distortion rules

Answer:

In our program the distortion is that every two consecutive letters are swapped, that is first and second, then third and fourth.

Code:

#include <stdio.h>
#include <stdlib.h>
#include <string.h>
int main()
{
    char message[20];
    printf("Enter message: ");
    scanf("%[^\n]c",message);
    printf("Distorted message is: ");
    int len=strlen(message);
    if(len%2==0)
    {
        for(int i=0;i<len;i=i+2)
            printf("%c%c",message[i+1],message[i]);
    }
    else
    {
        for(int i=0;i<len-1;i=i+2)
            printf("%c%c",message[i+1],message[i]);
        printf("%c",message[len-1]);
    }
    return 0;
}
