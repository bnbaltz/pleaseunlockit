WalkingBot

This class represents the brain of the walking robot and controls everything the robot will be able to do. 

 

Important Information

A WalkingBot is only capable of moving in 4 directions -- it cannot move diagonally!
(The engineers are working hard to improve this.)

 

Fields and Constructors

A WalkingBot should keep track of its name, energy, and current Location.
Energy is represented as an integer from 0 to 100, with 0 being out of energy.
There should not be a default constructor. All WalkingBots start with 0 energy.

Regular parameterized constructor 

Methods

A WalkingBot should have the following methods:

  • getName() returns its name

  • getEnergy() returns its energy

  • getLocation() returns its Location

  • recharge() fills its energy to 100

  • getDistance(Location) returns the Manhattan distance to the given Location

    • Hint: Call the method you wrote for Location!

  • getDistance(WalkingBot) returns the Manhattan distance between itself and the input WalkingBot

  • walkTo(Location) will tell the WalkingBot to move to the new Location. It will announce each step it takes. Make sure to update its Location after it moves!
    Each step takes 1 energy. If the WalkingBot does not have enough energy to get to the Location, then it will not move at all.

    • Example: A WalkingBot at (0,0) is trying to move to (1,2). It prints
      (0,0) (1,0) (1,1) (1,2)

    • Example: A WalkingBot at (0,0) is trying to move to (1,2) with 1 energy. It prints “I do not have enough energy to get to (1,2)”.

import java.lang.*;

//defining class Walking Bot
class WalkingBot{
    // definig params
    private String name;
    private int energy;
    private int[] location = new int[2];
    
    
    // parameterized constructor
    public WalkingBot(String name, int[] location){
        this.name = name;
        this.location = location;
    }
    
    // getter functions
    // getName
    public String getName(){
        return this.name;
    }
    
    // function getEnergy
    public int getEnergy(){
        return this.energy;
    }
    
    // function get location
    public int[] getLocation(){
        return this.location;
    }
    
    // function recharge
    public void recharge(){
        // seting energy to 100
        this.energy = 100;
    }
    
    // function getDistance return Manhattan distance for location
    public int getDistance(int[] location){
        // returning Manhattan distance
        return Math.abs(location[0]-this.location[0]) + Math.abs(location[1]-this.location[1]);
    }
    
    // function getDistance return Manhattan distance for another bot
    public int getDistance(WalkingBot bot){
        // fetching location of bot
        int[] location = new int[2];
        location = bot.getLocation();
        // calling getDistance for location
        return this.getDistance(location);
    }
    
    // function walkTo
    public void walkTo(int[] location){
        // fetching distance
        int distance = this.getDistance(location);
        
        // checking for enough energy
        if(this.energy>=distance){
            // walking in x
            int x_diff = location[0]-this.location[0];
            // x increment
            int x_inc = x_diff/Math.abs(x_diff);
            
            // looping for x increments
            for(int i=0;i<Math.abs(x_diff);i++){
                // displaying current location
                System.out.print("("+this.location[0]+","+this.location[1]+") ");
                
                // incrementing x loc by x_inc
                this.location[0]+= x_inc;
                
                // decrementing energy by 1
                this.energy -= 1;
            }
            
            // walking in y
            int y_diff = location[1]-this.location[1];
            // x increment
            int y_inc = y_diff/Math.abs(y_diff);
            
            // looping for x increments
            for(int i=0;i<Math.abs(y_diff);i++){
                // displaying current location
                System.out.print("("+this.location[0]+","+this.location[1]+") ");
                
                // incrementing x loc by x_inc
                this.location[1]+= y_inc;
                
                // decrementing energy by 1
                this.energy -= 1;
            }
            // displaying final location
            System.out.print("("+this.location[0]+","+this.location[1]+") ");
        }
        else{
            System.out.println("I do not have enough energy to get to ("+location[0]+","+location[1]+")");
        }
    }
    
    
}


// Main Driver Class
public class Main{
     // Main Function
     public static void main(String []args){
        // location array
        int[] location = {0,0};
        // object of WalkingBot
        WalkingBot bot1 = new WalkingBot("Bot1",location);
        // recharging bot
        bot1.recharge();
        
        // walk to location 1,2
        int[] walkToLocation = {1,2};
        // calling method walkTo
        bot1.walkTo(walkToLocation);
     }
}
