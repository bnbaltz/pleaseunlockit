QUESTION:

Define a set X of numbers as follows.
B. 2 X.
R1. If x X, so is 10x.
R2. If x X, so is x + 4.
List all the elements of X that are less than 30.

ANSWER:

Since 2 is in X,
10*2 =20 is in X
2+4 = 6 is in X

now for each of those, 10x and x+4 must also be in X
and then for each of those, 10x and x+4 must also be in X
We continue this pattern.

All the ones under 30 are:
2,6,10,14,18,20,22,24,26,28
