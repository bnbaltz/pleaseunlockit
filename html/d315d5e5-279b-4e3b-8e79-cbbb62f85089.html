(A)

Work done in thermal process : 

W = nRTinitialln(Vfinal/Vinitial)

For the expansion process

W1 =( 0.3 mol)*(8.3145)*(800K)ln(6000/1000)

    = 3575.42 J 

For compression process

W3= (0.3 mol)*(8.3145)*(300 k)ln(1000/6000)

     = -1340.78 J

Net work done by the engine per cycle : 

W = W1+W2+W3+W4

   = 3575.42 J + 0 J -1340.78 J + 0 J

  = 2234.64 J

(B)

Thermal efficiency : 

= W/Qin

Qin= Q1+Q4

Q1= W1= 3575.42 J

Q4= (3/2)*(0.3 mol)*(8.3145)*(800k-300k)

   = 1870.76 J

Qin= Q1+Q4= 3575.42J+1870.76 J = 5446.18 J

Hence 

= W/Qin=2234.64J/5446.18J = 0.41  or 41%
