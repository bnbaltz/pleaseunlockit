Question 7: Based on standard curve ,calculate the extinction coefficient of pNP?

Answer: It can be calculated using formula ,

Concentration = Abs x Extinction coefficient x path length

So, Extinction coefficient = Concentration / Abs x path length

Concentration of pNP = 0.08 at Abs = 1.82, path length = 1 cm, putting these value in the formula we get,

Extinction coefficient = Concentration / Abs x path length

Extinction coefficient = 0.08 / 1.82 x 1 = 0.043 M-1 cm-1

Extinction coefficient of pNP =  0.043 M-1 cm-1 ( if path length = 1 cm)
