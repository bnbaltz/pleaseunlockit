Formic acid = HCOOH

Sodium formate = HCOONa

Given that total volume of buffer = 25 mL

   Let us assume , volume of HCOONa = X ml

then volume of HCOOH = 25- X ml

Given that [HCOOH]/[[HCOONa] = 3.4 ------------ Eq (1)

[HCOONa] = molarity x volume = 0.1 M * X mL = 0.1X mmol

[HCOOH] = molarity x volume = 0.1 M x (25-X)  mL = (0.1) (25-X) mmol

From eq (1),

[HCOONa]/[[HCOONa] = 3.4

0.1X mmol/(0.1) (25-X) mmol = 3.4

0.1X = 8.5-0.34X

0.44X = 8.5

X = 19.3 mL

X is volume of HCOONa( sodium formate).

sodium formate required = 19.3 mL
