QUESTION:

In this assignment, students will calculate the annual rate of growth needed for BZU system enrollment to reach a specified target.

  1. Create the following variable to use in calculating enrollment scenarios
    initial_enrollment  29,933 (enrollment in 2019)

 Note: The language should be C

ANSWER:

First Question a, b, c and d answer:-

Code is:- (In C)

#include <stdio.h>
#include <math.h>

int main() {
    
    int target, flag, temp;
    int ini = 29933;     // enrollment in 2019 
    float growth;
    
    printf("Enter enrollment target for 2030: ");
    flag = scanf("%d", &target);
    while(flag != 1){
        while((temp=getchar()) != EOF && temp != '\n');
        printf("Please enter valid target: ");
        flag = scanf("%d", &target);
    }
    
    growth = (pow((float)target / ini, 1.0/11.0) - 1) * 100.0;
    
    printf("Annual rate of growth: %.1f%%\n", growth);
    
    if(growth < 0)
        printf("Negative growth rate");
    else if(growth >= 0 && growth < 1)
        printf("Reasonable growth rate");
    else if(growth >= 1 && growth < 2)
        printf("Ambitious growth rate");
    else if(growth >= 2 && growth < 4)
        printf("Very ambitious growth rate");
    else
        printf("Unreasonable growth rate");
    
    return 0;
}
