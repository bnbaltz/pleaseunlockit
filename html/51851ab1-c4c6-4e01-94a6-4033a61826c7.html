QUESTION:

write a complete program that uses a FOR loop to display the number in the interval [-100, 350] in increments of 5.

JAVA Pleas

ANSWER: 

public class NumberInterval {

    public static void main(String[] args) {

        int start = -100 , stop=350;

        for(int i = start ; i<stop ; i=i+5)
        {
            System.out.printf("%d  ",i);
        }
    }
}
