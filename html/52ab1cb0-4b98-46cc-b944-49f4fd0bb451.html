Annual payment = $12,600
Number of payments = 13
Interest rate = 7.50%

If payments are made at the end of each year:

Present value = $12,600/1.075 + $12,600/1.075^2 + … + $12,600/1.075^13
Present value = $12,600 * (1 - (1/1.075)^13) / 0.075
Present value = $12,600 * 8.12584
Present value = $102,385.58

If payments are made at the beginning of each year:

Present value = $12,600 + $12,600/1.075 + … + $12,600/1.075^12
Present value = $12,600 * 1.075 * (1 - (1/1.075)^13) / 0.075
Present value = $12,600 * 8.73528
Present value = $110,064.53

So, payments will worth $7,678.95 ($110,064.53 - $102,385.58) if each payment is made at the beginning of the year.
