QUESTION:

Python IDLE 3.9 Simple Function - Given a single parameter (number) write a function that tells how many times the digit 6 appears in the number.  The function should return the number of 6s found in the number.

ANSWER: 

# function that tells how many times the digit 6 appears in the number
def count6(n):
    # initializing result to 0
    result = 0

    # Looping till n becomes 0
    while n!=0:

        # Checking last digit of n is 6
        if(n%10==6):

            # Updating result
            result += 1

        # removing last digit
        n = n // 10

    # Returning result
    return result

# Testing
print(count6(12345678))
print(count6(12645678))
print(count6(12665678))
print(count6(12578))
