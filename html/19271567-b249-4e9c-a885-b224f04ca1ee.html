QUESTION:

Create an Arduino program that uses 2 sensors: Button Light sensor Every time the button is pressed, the intensity of the light changes. Cycle through the following 6 intensities: 0, 51, 102, 153, 204, 255. Use port D2 for the button and D3 for the light sensor.(please attach code and output)

ANSWER:

Arduino Uno

Atmega328p microcontroller is used in Arduino mega. So Arduino can be considered as a microcontroller based development board. There are 14 digital pins which allows the digital input/output devices to be interfaced with Arduino and 6 analog channels for analog devices such as potentiometer, sensors etc. 

Arduino Programming 

The basic arduino program consists of two functions 

void setup()

This function is used for the configuration of the modules used for the application. Digital pins , Serial communication modules are configured inside this function. This function will only be executed once. 

Syntax- 

void setup()

{

Configuration statements 

}

void loop()

This function is used for executing the operations to be performed using the controller. As the name suggests, this function will execute till the power down or reset. 

Syntax- 

void loop()

{

Operations to be performed 

}

pinMode() - This function is used to configure a digital pin as input or output according to the requirement.  

Syntax - 

pinMode(5,OUTPUT) - This statement configures digital pin 5 as output 

pinMode(6,INPUT) - This statement configures digital pin 6 as input   

digitalRead() - This function read the state of a digital pin. This function will return either 1 or 0 according to the pin state. 

Syntax-  

digitalWrite(5) - This statement read the state of digital pin 5

analogWrite()

This function is used to generate PWM signal of different duty cycle on PWM pins. PWM pins on Arduino Uno are 3,5,6,9,10 and 11. 

Syntax- 

analogWrite(pin,value) 

pin - PWM pin 

value - Value can be in range 0 -255, 0 means 0% duty cycle and 255 means 100% duty cycle.  

 

Here the brightness of the Light which is an LED must be controlled by a button. So the LED is connected to PWM pin 3, and analogWrite() function can be used to control the brightness.  

Circuit diagram 

 

The code will be, 

int count=0;                             // Initially count variable is zero
void setup()
{
  pinMode(3, OUTPUT);                    // Configure LED pin as output and Button pin as input 
  pinMode(2, INPUT);
  
}

void loop()
{
  if(digitalRead(2)==1 && count==0)     // If the button is pressed for first time    
  {
    while(digitalRead(2)==1);           // Wait till released 
    analogWrite(3,0);                   // LED will be off since 0 is written 
    count++;                            // Increment count 
    
  }
  
  if(digitalRead(2)==1 && count==1)     // If the button is pressed for second time
  {
    while(digitalRead(2)==1);           // Wait till released 
    analogWrite(3,51);                  // LED will be ON but with less brightness 
    count++;                            // Increment count 
  }
  
  if(digitalRead(2)==1 && count==2)     // If the button is pressed for third time
  { 
    while(digitalRead(2)==1);           // Wait till released
    analogWrite(3,102);                 // LED will be ON  with more brightness
    count++;                            // Increment count 
  }
  if(digitalRead(2)==1 && count==3)     // If the button is pressed for fourth time
  { 
    while(digitalRead(2)==1);           // Wait till released  
    analogWrite(3,153);                 // Increase the brightness 
    count++;                            // Increment the count 
  }
  
  if(digitalRead(2)==1 && count==4)     // If the button is pressed for fifth time
  {
    while(digitalRead(2)==1);           // Wait till released
    analogWrite(3,204);                 // Increase the brightness 
    count++;                            // Increment the count 
  }
  
  if(digitalRead(2)==1 && count==5)     // If the button is pressed for sixth time
  { 
    while(digitalRead(2)==1);           // Wait till released
    analogWrite(3,255);                 // Maximum brightness 
    count=0;                            // Initialize count to 0
  }
  
  
}
