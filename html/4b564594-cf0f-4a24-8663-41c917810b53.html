5) R13=0x34BD5D98

   in cortex M4 each register is of 32 bit size, so after each push register memory in R13 is decreased by 4.

 So the value of R13 after Push command is 0x34BD5D8C.

value  of R13 after PUSH command is 0x2B592E4
