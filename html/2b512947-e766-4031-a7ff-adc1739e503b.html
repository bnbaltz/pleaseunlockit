Consider the home security system with the following inputs:

• 6 sets of sensor switches

• Master switch

• Alarm switch

• Light switch

• Police switch

The outputs to be delivered are:

• A : alarm ( 0 = alarm on, 1 = alarm off )

• L : light ( 0 = lights on, 1 = lights off )

• V : video cameras ( 0 = video cameras off, 1 = video cameras on )

• C : call to police ( 0 = call off, 1 = call on )

The logical gates to be used are:

• The AND gate produces the output 1 when both the inputs are 1. Otherwise, it returns the output 0.

• The OR gate produces the output 0 when both the inputs are 0. Otherwise, it returns the output 1.

• The NOT gate inverts the value 0 to 1 and vice versa.

The security system of a home is designed as follows:

Master switch:

• The master switch detects an intrusion when one or any one of the six sets of the sensors detects an intrusion. The inputs of six sensors are connected to NOR gate. Because, it generates output 1 when only all inputs are 0 and output 0 for anyone or all inputs are 1.

• The master switch enables security alert only when the sensor sets detects intrusion. As, the output of the sensor sets is 0, the master switch is also enabled with signal value 0.

1. As the security system is enabled, the alarm signal is also used as input with signal value 1. Here, the output of AND gate is inverted in order to enable the Alarm ‘A’.

2. As the security system is enabled, the lights are also enabled and used as input with signal value 1. Here, the output of AND gate is inverted in order to enable the lights ‘L’.

3. As the security system is enabled, the police switch is also enabled and used as input with signal value 1. Here, the output of AND gate enables the call to police ‘P’.

4. Finally, the output of AND gate for sensor sets and master switch is inverted in order to enable the Video cameras ‘V’.
