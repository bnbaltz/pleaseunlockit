QUESTION:

Write a program that implements the pseudocode below:

Ask the user for their taxable income.
Compute and output the marginal tax rate as follows:
  -  Error if income is negative
  -  10% if income $0 - $9,700
  -  12% if income $9,701 – $39,475
  -  22% if income $39,476 – $84,200
  -  24% if income $84,201 – $160,725
  -  32% if income $160,726 – $204,100
  -  35% if income $204,101 – $510,300
  -  37% if income $510,301+

Think about the order of your decisions: elif/else clauses are considered only if the previous condition is false!

A sample run of your program should look like:

Enter taxable income: 120000
Marginal tax rate: 24%

and another sample run:

Enter taxable income: -50000
Marginal tax rate: Error

 

 

Must be done in python idle

income = int(input())

    m_tax = 22

    m_tax = 24
elif(income>=160726 and income<=204100):
    m_tax = 32
elif(income>=204101 and income<=510300):
    m_tax = 35
elif(income>=510301):
    m_tax = 37
else:
    m_tax = 0
#condition for verifying if m_tax is greater than zero

    print("Marginal tax rate: {}%".format(m_tax))
else:
    print("Marginal tax rate: ERROR")
