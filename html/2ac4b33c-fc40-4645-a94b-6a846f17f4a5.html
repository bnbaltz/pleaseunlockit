QUESTION:

Python IDLE 3.9 - Simple Functions - Given a number and a secret number - write a function that tells the relationship between the numbers. The function should return “Higher” if number < secretNumber, “Lower” if number > secretNumber, and “Correct” if number == secretNumber.

ANSWER:

def tell(number, secretNumber):

    if number < secretNumber:

        print("Lower")

    elif number > secretNumber:

        print("Higher")

    else:

        print("Correct")

 

tell(123, 1234)
