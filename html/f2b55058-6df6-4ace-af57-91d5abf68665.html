QUESTION:

Jefferson & Daughter has a cost of equity of 14.6 percent and a pre-tax cost of debt of 7.8 percent. The required return on the assets is 13.2 percent. What is the firm's debt-equity ratio based on M&M II with no taxes?

0.26

0.33

0.37

0.43

0.45

ANSWER:

Solution:

As per the Modigilani and Miller ( MM ) Model the formula for calculating the cost of equity is :

ke = ROA + [ ( ROA − kd) * ( D/E ) ]

Where

ke = Cost of Equity ;    ROA = Required Return on Assets  ;   kd = Post Tax Cost of debt    ;  D/E  = Debt Equity Ratio

( Since there are no taxes the Pre tax cost of debt will be equal to Post Tax cost of Debt )

 

As per the information given in the question we have

ke = 14.6 %  ;    ROA = 13.2 %   ;   kd = 7.8 %     ;  D/E  = To find

 

Applying the above values in the formula we have

14.6 = 13.2 + [ ( 13.2 – 7.8 ) * ( D/E ) ]

14.6 - 13.2 =  [ ( 13.2 – 7.8 ) * ( D/E ) ]

1.4 = [ ( 13.2 – 7.8 ) * ( D/E ) ]             

1.4 = [ ( 13.2 – 7.8 ) * ( D/E ) ]             

1.4 = 5.4 * ( D/E )

D/E = 1.4 / 5.4 = 0.2593

D/E = 0.26 ( when rounded off to two decimal places )

 

Thus the firm's debt-equity ratio based on M&M II with no taxes = 0.26

The solution is Option 1 = 0.26
