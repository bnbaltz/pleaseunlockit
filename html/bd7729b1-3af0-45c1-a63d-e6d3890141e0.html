Binary coded decimal:

The binary coded decimal is used to store the decimal number in to the binary form.

The binary number requires 4 bits to represent the decimal number 0 to 9.

To design a gray code-to-BCD code there are some steps which should be followed. These steps are as follows:

• Truth table

• Karnaugh map(k-map)

• Circuit

4 bits will be used to represent the BCD as input. Valid BCD for 4 bits comprises 0 to 9. For any value more than that, 0000 will be output to denote invalid BCD input. 0000 was chosen for this purpose because it is not used for output of any valid BCD input.

Truth table:

In the given table inputs are the in BCD encoding (0 to 9) and outputs are the binary number with addition of 3 (input +3 = output). After input = 9, the output becomes invalid as 4 bit BCD can accommodate only from 0 to 9.

Represent the invalid output as “don’t care”.
