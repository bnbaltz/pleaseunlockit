QUESTION:

A certain type of aluminum screen has, on the average, one flaw in a 100-foot roll. Assuming that the number of flaws has a Poisson distribution, find the probability that a 150-foot roll has at most one flaw

ANSWER:

Here, λ = 1.5 and x = 1
As per Poisson's distribution formula P(X = x) = λ^x * e^(-λ)/x!

We need to calculate P(X <= 1).
P(X <= 1) = (1.5^0 * e^-1.5/0!) + (1.5^1 * e^-1.5/1!)
P(X <= 1) = 0.2231 + 0.3347
P(X <= 1) = 0.5578
