As per Laplace criteria:

  1. Equal probability will be assigned to each of the 3 outcomes i.e. Low demand , Medium demand and High demand. Therefore, probability of each option will be 1/3
  2. Expected value for each option ( i.e. small bus, medium bus and large bus ) will be calculated as Sum of   (  probability x corresponding profit for each type of demand)

 

Therefore ,

 

Expected profit for Small Bus option = 1/3 x ( 50,000 + 60,000 + 70,000 ) = 1/3 x 180,000 = 60,000

Expected profit from Medium Bus option = 1/3 x ( 40,000 + 80,000 + 90,000) = 1/3 x 210,000 = 70,000

Expected profit from Large bus option = 1/3 x ( 20,000 + 50,000 + 120,000) = 1/3 x 190,000 = 63,333.33

 

  1. Choose the option with highest expected profit.

There option to be chosen is “Medium Bus” which has the highest expected profit of $70,000
