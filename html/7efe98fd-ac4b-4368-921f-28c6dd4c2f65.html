QUESTION:

Calculate the energy (in eV/atom) for vacancy formation in some metal, M, given that the equilibrium number of vacancies at 437oC is 2.72E+23 m-3. The density and atomic weight for this metal are 13.3 g/cm3 and 75.30 g/mol, respectively.

ANSWER:

All parameters besides Qv are given except N, the total number of atomic sites.  However, N is related to the density, (?M), Avogadro's number (NA), and the atomic weight (AM) according to 

N = NA?M / AM

   = (6.023 x 10 23) x 13.3 / (75.3) 

   = 1.063 x 10 23 atoms / cm3 = 1.063 x 1029 atoms / m3 

 = - (8.62 x 10-5 eV / atom - K) (437 + 273 K) ln (2.72 x 1023 / 1.063 X 1029)

 = - (8.62 x 10-5 eV / atom - K) (710 K) (-12.87)

 = 0.79 eV / atom
