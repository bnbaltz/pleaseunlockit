QUESTION:

How to do the following by using the Eclipse Java debugger

1. look up the value of a variable

2. Change the value of a variable in the debugger.

3. Step through your program one statement at a time.

4. Seting break points in your program.

ANSWER:

1. To look up the value in eclipse debugger you can use expression windows. While debugging, go to menu window and then show view then goto expressions there you can see the contents.

2. You can access variables that means you can change the variable content through variable view there you can right click on any variable and select change the value and then you can change it.

3.  Stepping through the execution of a java program in eclipse line by line. If a breakpoint is kept while performing any operation the execution is suspended at the breakpoint and step operation is ended.

4. To keep breakpoints in your program in eclipse double click on the margin of thge line you want to stop the execution, then you press Shift + ctrl + B. We have f5, f6 f7 to control the execution like step into, step over and step return buttons.
