QUESTION:

When two capacitors are connected in series, the equivalent capacitance of the combination is 120 µF. When the two are connected in parallel, however, the equivalent capacitance is 480 µF. What are the capacitances of the individual capacitors?


ANSWER:

Let the capacitances be C and C '

The equivalent capacitance when they are in series combination = 120 µF

                                                                              CC' /(C+C') = 120 μF

The equivalent capacitance when theyare connected in parallel = 480 µF

                                                                                   C + C' = 480 μF        ----( 1)

From above two equations  CC' = 120 (C+C')

                                             = 120 μF x 480 μF

We know  ( C-C') 2 = (C+ C') 2 -4CC'

                           = 230400 - 230400

                           = 0

                C - C ' = 0

                      C = C '

Substitute this in eq( 1) we get  C + C = 480 μF

                                                     2C = 480 μF

                                                       C = 240 μF

So, Capacitances are C = 240 μF

             and C ' = 240 μF
