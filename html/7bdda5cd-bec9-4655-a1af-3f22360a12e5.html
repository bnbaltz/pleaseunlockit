The given IP address is 70.70.7.0  /8

The /8 notation indicates # of network ID bits is 8 and therefore number of host ID bits is 24

For creating subnet we need to borrow bits from host ID bits. 

So, the subnet bits required is 1.

For subnet address, all the host bits will be all 0. For broadcast address, all host bits will be all 1.
