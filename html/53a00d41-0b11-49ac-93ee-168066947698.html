Formulating set of equations,

Balancing horizontal and vertical forces on each node,

for top node,

F1Cos30 - F3Cos60 = 0 ...........(i)

F1Sin30 + F3Sin60 = -1000 .........(ii)

For below left node,

F1Sin30 + V2= 0...................(iii)

F2+ H2+ F1Cos30=0 .............(iv)

For bottom right node,

F2+ F3Cos60 = 0 ..................(v)

F3Sin60 + V3= 0 ....................(vi)

Now ,we have 6 equations and 6 variables which can be easily solved in matlab if we will put it in matrix form as

AX=B

Where,

X=Matrix of all 6 unknown forces

So,In matrix,

A=[Cos60 ,0, -Cos60, 0,0. 0;Sin30,0,Sin60,0,0,0;Sin30,0,0,1,0,0;Cos30,1,0,0,1,0;0,1,Cos60,0,0,0;0,0,Sin60,0,0,1];

B=[0;-1000;0;0;0;0]

X=inv(A)*B

result X will be a 6x1 matrix in the sequence,[F1;F2;F3;V2;H2;V3]
