Sharpe ratio =Excess return on the portfolio / volatility on the portfolio

                     =( E(r) - Rf ) / standard deviation

E(r) = Expected return

Rf = Risk free rate 

Sharpe ratio of portfolio b = ( 7 - 5) / 10

                      = 2 / 10

                      = 0.20

Sharpe ratio of portfolio c = (10 - 5) / 20

                                             = 5 / 20

                                             = 0.25

Sharpe ratio of portfolio d = (14 - 5 ) / 40
