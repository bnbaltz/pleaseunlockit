b) The Dickey Fuller test used to test for stochasticity can be applied to the test the difference Y – C. That is to test the regression

Using the null hypothesis, if the null is not rejected, then Y – C is stochastic and non-stationary. However, if the null is rejected, then Y – C is stationary.
